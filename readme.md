## install libraries

install vite to have a local dev server that will refresh the browser on code change.

```{bash}
npm install -g vite
```

install libraries

```{bash}
npm install
```

## start server

```{bash}
cd <directory>
vite
```

## generate new directory

copy `template` directory

```{bash}
./generate_files.sh <new directory>
```
